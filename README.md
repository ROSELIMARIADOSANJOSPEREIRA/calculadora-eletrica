Calculadora de Consumo Elétrico 💡

Este é um script simples em **Python** desenvolvido para calcular o consumo mensal de energia de eletrodomésticos e o custo estimado na conta de luz.

⚙️ Como Funciona
O programa solicita três informações ao usuário 😎👾:
1. O nome do aparelho (ex: Notebook, Geladeira).
2. A potência do aparelho em Watts (W).
3. A quantidade de horas que o aparelho fica ligado por dia.

Com esses dados, o script calcula o consumo mensal em **kWh** e o valor em **Reais (R$)**, baseando-se em um mês de 30 dias e uma tarifa de R$ 0,75/kWh.

 Tecnologias Utilizadas
 
 ![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
 
 Python 3: Linguagem principal.


 Exemplo de Execução
Ao rodar o `app.py`, o resultado será algo como:
- **Aparelho:** Notebook
- **Potência:** 140W
- **Uso:** 12h/dia
- **Resultado:** R$ 37,80/mês
